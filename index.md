## 学习笔记

  >**Java开发学习**  

  - Java 开发是以 Java 编程语言为核心的软件开发技术，由 Sun Microsystems（后被 Oracle 收购）于 1995 年推出，核心设计理念是 “一次编写，到处运行（Write Once, Run Anywhere，WORA）”。Java 程序编译后生成与平台无关的字节码文件，通过不同操作系统的 Java 虚拟机（JVM）解释执行，无需针对不同系统重新编译，这是其最核心的优势。  

  - Java 完全遵循面向对象编程（OOP）思想，封装、继承、多态三大特性让代码更易维护和复用。它内置垃圾回收机制（GC）自动管理内存，搭配安全管理器、字节码校验等机制，保障程序的安全性与稳定性。同时 Java 拥有丰富的生态，涵盖海量标准库和第三方框架，比如 Spring、MyBatis、Spring Boot 等。  

  - Java 开发的应用场景十分广泛，是金融、电商、政务等领域企业级后端系统的首选技术；也是安卓应用开发的基础语言；Hadoop、Spark 等主流大数据框架均基于 Java 开发，使其成为大数据开发的必备技能；此外还可用于嵌入式开发和中间件开发等场景。  

  - Java 开发的基础流程为：编写 Java 源代码文件，通过 javac 编译器编译为字节码文件，再由 JVM 加载并执行字节码。新手入门需掌握基础语法、核心类库、开发工具（IDEA/Eclipse）、构建工具（Maven/Gradle）以及入门框架（Spring Boot）等核心技术栈。

  >**PHP开发学习**

  - PHP（Hypertext Preprocessor，超文本预处理器）是一种开源的服务器端脚本语言，专为 Web 开发设计，1994 年由 Rasmus Lerdorf 最初创建，如今已成为全球最主流的 Web 后端开发语言之一。
  
  - PHP 最大的特点是与 HTML 高度兼容，代码可直接嵌入 HTML 文档中执行，无需额外编译步骤，服务器端解析后直接生成动态 HTML 页面返回给浏览器，开发效率极高。它支持几乎所有主流数据库（MySQL、PostgreSQL、Oracle 等），尤其与 MySQL 组合是中小型网站开发的经典搭配，搭建成本低、上手门槛低。
  
  - PHP 具备跨平台特性，可运行在 Windows、Linux、Unix 等操作系统上，兼容 Apache、Nginx、IIS 等主流 Web 服务器，还支持多种协议（HTTP、FTP、SMTP 等），能轻松实现表单处理、会话管理、文件上传、数据交互等 Web 开发核心需求。
  
  - PHP 的生态体系成熟，拥有丰富的内置函数库，覆盖字符串处理、数组操作、网络请求、加密解密等各类场景；主流框架如 Laravel、ThinkPHP、Yii 等，解决了原生 PHP 代码复用性低、架构松散的问题，可支撑从个人博客到电商平台、企业官网等不同规模的 Web 项目开发。
  
  - PHP 开发的核心流程：将嵌入 PHP 代码的 .php 文件部署到支持 PHP 解析的 Web 服务器，当用户访问对应 URL 时，服务器解析 PHP 代码并与数据库 / 文件系统交互，生成动态 HTML 内容返回给客户端浏览器。新手入门需掌握 PHP 基础语法、变量与数据类型、流程控制、数据库操作（PDO/MySQLi）、超全局变量，以及一款主流框架的基础使用。
